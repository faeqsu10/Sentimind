# Sentimind 사용자 참여 기능 기획서

## 리서치 기반
- 4개 전문가 에이전트 병렬 리서치 (심리학, 시각화, UX, 학술 논문)
- 35개 논문/연구 참조
- 5개 경쟁 앱 분석 (Daylio, Moodflow, Pixels, Reflectly, Finch)

## 핵심 발견

### 심리학 근거
- **자기결정이론(SDT)**: 자율성(프롬프트 선택), 유능감(성장 트래커), 관계성(AI 공감)
- **긍정심리학(PERMA)**: 감사 일기가 삶의 만족도 6.86% 향상 (Boggiss 2023)
- **습관 루프**: CUE(리마인더) → ROUTINE(짧은 입력) → REWARD(AI 공감 + 시각화)
- **감정 라벨링 자체가 치료적 효과** (MeMO Study, Frontiers 2021)

### 경쟁 분석
- Sentimind 독점 강점: AI 실시간 공감 메시지 (경쟁 4사 미보유)
- 4대 공백: Year in Pixels, 위젯, 맞춤형 감정 레이블, 활동 태깅

### 리텐션 통계
- 정신건강 앱 D30 리텐션 중앙값: 6.1% (Baumel 2019)
- 리더급(Daylio) D30: 18%
- 이탈 원인 81%: 습관 미형성 + 인사이트 부재 + 입력 마찰

## Phase 1 구현 (즉시) — 현재 진행 중

### F1. 감정 꽃잎 차트 (마음의 꽃잎)
- **근거**: Plutchik 감정 바퀴, 감정 시각화 참여도 연구
- **설명**: 감정 분포를 방사형 꽃잎 SVG로 시각화
- **위치**: 통계 탭 > 감정 패턴 인사이트 앞
- **복잡도**: 중

### F2. 감사 일기 프롬프트 모드
- **근거**: Boggiss 2023 (만족도 +6.86%), Viehl 2022 (한 줄도 효과)
- **설명**: 5가지 글쓰기 방향 칩 (자유, 감사, 작은 기쁨, 셀프 토크, 주간 회고)
- **위치**: 일기 입력 textarea 위
- **복잡도**: 낮

### F3. 감정-상황 교차 히트맵
- **근거**: 교차 분석, 온톨로지 활용
- **설명**: 감정(행) × 상황(열) 빈도 히트맵
- **위치**: 통계 탭 > 상황 차트 뒤
- **복잡도**: 중하

## 설계 원칙 (연구 기반)
1. 주 2-3회 사용이 최적 빈도
2. 15-20분이 이상적 세션 길이
3. 외적 보상(스트릭) → 내적 보상(인사이트)으로 점진 전환
4. GenAI 기반이 규칙 기반보다 효과적 (JMIR 2025)
5. 안전장치 필수 (면책 고지, 위기 감지)
6. 짧은 형식(한 줄)도 스트레스 감소에 유의한 효과
