## 5. 월간 비용 추정

### 가정

- 일 100명 사용자, 1인 1회 분석 요청
- 월 3,000회 요청
- 평균 입력: 50 토큰 (한 줄 일기 + 시스템 프롬프트 약 500 토큰 = 550 토큰)
- 평균 출력: 150 토큰 (감정 분석 JSON 응답)
- 월간 총 토큰: 입력 1.65M, 출력 0.45M

| 모델 | 월 입력 비용 | 월 출력 비용 | 월 총 비용 | 추천도 |
|------|------------|------------|-----------|--------|
| gemini-2.5-flash | $0.000495 | $0.001125 | **$0.0016** | 최적 |
| gemini-2.5-flash-lite | $0.000165 | $0.000180 | **$0.0003** | 비용 최적화 |
| gemini-2.5-pro | $0.002063 | $0.004500 | **$0.0066** | 과잉 |
| gemini-3.1-flash-lite | $0.000413 | $0.000675 | **$0.0011** | Preview |

참고: 일 3,000건 수준에서는 **모든 모델이 사실상 무료에 가까운 비용**입니다.

### 프리 티어만으로 운영 가능성

| 모델 | 일 요청 한도 (RPD) | 일 100건 가능? | 일 최대 사용자 |
|------|-------------------|---------------|--------------|
| gemini-2.5-flash | 250 RPD | 가능 | ~250명 |
| gemini-2.5-flash-lite | 1,000 RPD | 가능 | ~1,000명 |
| gemini-2.5-pro | 100 RPD | 가능 | ~100명 |

**결론: 일 250명 이하 사용자라면 gemini-2.5-flash 프리 티어만으로 충분합니다.**

## 6. 권장사항

### 현재 상태: gemini-2.5-flash -- 유지 권장

**이유:**

1. **가격 대비 성능 최적**: 한 줄 일기 감정 분석에 충분한 성능을 갖추면서 합리적인 가격
2. **Stable (GA)**: 프로덕션 안정성 보장, 장기 지원 예상
3. **프리 티어 충분**: 일 250건까지 무료 사용 가능
4. **한국어 성능**: 감정 뉘앙스를 잘 파악하며 JSON 구조 출력이 안정적
5. **Thinking 기능**: 필요 시 thinking budget을 활성화하여 복잡한 감정 분석 가능

### 향후 고려 사항

| 시나리오 | 권장 모델 | 이유 |
|---------|---------|------|
| 현재 (소규모) | gemini-2.5-flash | 최적의 균형 |
| 비용 극한 절감 | gemini-2.5-flash-lite | 1/6 비용, 프리 티어 4배 |
| 사용자 1,000명+ | gemini-2.5-flash (Tier 1) | 월 수십 원 수준의 유료 전환 |
| 최고 품질 필요 | gemini-2.5-pro | 비용 10배 증가, 성능 향상은 미미 |
| 3.x 안정화 후 | gemini-3-flash 계열 | GA 전환 시 평가 후 마이그레이션 |

### 모델 전환 시 변경 사항

`config/llm-config.js`에서 환경변수로 모델을 변경할 수 있습니다:

```bash
# .env에서 모델만 변경
GEMINI_MODEL=gemini-2.5-flash-lite

# flash-lite 사용 시 thinking은 지원되지 않을 수 있으므로 확인 필요
GEMINI_THINKING_BUDGET=0
```

### 추가 최적화 방안

1. **Context Caching**: 시스템 프롬프트(약 500 토큰)를 캐싱하면 반복 비용 절감
   - gemini-2.5-flash 캐시: $0.03/1M (일반 $0.30의 1/10)
   - 단, 캐시 저장 비용 $1.00/시간이므로 트래픽이 충분해야 손익분기

2. **Batch API**: 일기 분석이 실시간이 아닌 경우 50% 할인 적용 가능
   - 현재 프로젝트는 실시간 응답이 필요하므로 해당 없음

3. **Rate Limit 설정**: 현재 8 req/min으로 설정됨 (프리 티어 10 RPM 이하)
   - flash-lite 전환 시 15 RPM까지 가능하므로 `RATE_LIMIT_MAX=12`로 상향 가능

## 7. 결론

**gemini-2.5-flash를 현재 그대로 유지하는 것이 최선입니다.**

- 한 줄 일기 감정 분석이라는 작업 특성상 Pro급 모델은 과잉
- Flash-lite는 비용 절감에 유리하나 감정 뉘앙스 파악 품질이 다소 떨어질 수 있음
- 2.0 시리즈는 곧 종료되므로 절대 사용 불가
- 3.x 시리즈는 Preview 단계이므로 GA 전환 후 평가 권장
- 프리 티어만으로 일 250명까지 무료 운영 가능
