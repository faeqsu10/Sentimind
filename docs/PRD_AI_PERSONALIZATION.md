# PRD: AI 개인화 확장

**문서 버전**: v0.1
**작성일**: 2026-03-11
**상태**: Draft
**관련 문서**: `docs/PRD.md`, `docs/API.md`, `docs/DATABASE_SCHEMA.md`

## 1. 목표

Sentimind의 AI 응답을 사용자가 더 잘 맞는 방식으로 받을 수 있게 개인화 옵션을 확장한다. 이번 범위는 `표현 방식의 개인화`에 한정하며, 감정 분석 기준이나 안전 정책 자체는 바꾸지 않는다.

핵심 목표:
- 재방문율과 응답 만족도 개선
- 개인화 체감 강화
- 브랜드 톤을 해치지 않는 범위의 선택지 제공

이번 범위에 포함되는 기능:
- AI 응답 스타일 설정
- 제한된 프리셋 페르소나
- 유쾌한 톤 실험
- 텍스트 기반 3컷 그림일기 카드

## 2. 문제 정의

현재 사용자는 `ai_tone`만 선택할 수 있고, 응답 길이·조언 강도·관계형 페르소나를 제어할 수 없다. 그 결과 같은 분석 결과라도 “내 취향에 맞게 반응한다”는 느낌이 약하다.

또한 사용자 피드백 기준으로는 아래 수요가 확인됐다.
- AI 말투나 기능을 조정하고 싶다
- 오늘 하루를 그림일기처럼 정리받고 싶다
- 조금 더 유쾌한 말투를 원한다
- 특정 관계형 페르소나를 적용하고 싶다

## 3. 제품 원칙

- 개인화는 허용하지만 희화화는 제한한다.
- 페르소나는 표현만 바꾸고 판단 기준은 바꾸지 않는다.
- 위기성 표현, 강한 부정 감정에서는 항상 안전한 기본 톤으로 복귀한다.
- 실존 인물, 상담사, 의사, 유명인 흉내는 금지한다.
- 그림일기는 사실 단정이 아니라 분위기 기반 요약으로 안내한다.

## 4. 범위

### 4.1 In Scope

1. `AI 응답 스타일`
- 말투: `warm`, `calm`, `direct`, `playful`
- 응답 길이: `short`, `balanced`, `detailed`
- 조언 강도: `comfort`, `balanced`, `actionable`

2. `프리셋 페르소나`
- `gentle_friend`
- `calm_coach`
- `clear_reflector`

3. `유쾌한 톤`
- 기본 기능이 아니라 실험 옵션
- 민감 감정에서는 자동 비활성화

4. `3컷 그림일기 카드`
- 실제 이미지 생성 없음
- 제목, 3개 장면 설명, 장면별 캡션, 대표 감정, 마무리 회고 생성

### 4.2 Out of Scope

- 실존 인물/캐릭터 모사
- 이미지 생성 모델 연동
- 음성/아바타형 AI
- 페르소나별 별도 감정 분석 로직

## 6. 기능 요구사항

### FR-AIP-001 AI 응답 스타일 저장

- 프로필에서 말투, 응답 길이, 조언 강도를 저장할 수 있어야 한다.
- 저장값은 이후 `/api/analyze` 응답 생성에 반영되어야 한다.

수용 기준:
- 프로필 저장 후 다음 분석부터 선택한 스타일이 반영된다.
- 잘못된 enum 값은 `400 VALIDATION_ERROR`를 반환한다.

### FR-AIP-002 안전한 페르소나 프리셋

- 사용자는 제공된 페르소나 프리셋 중 하나를 선택할 수 있어야 한다.
- 페르소나는 문체와 대화 방식만 바꾼다.

수용 기준:
- `persona_preset`은 미지정 시 `none`으로 동작한다.
- 위기성 표현에서는 페르소나 지시보다 안전 지시가 우선한다.

### FR-AIP-003 유쾌한 톤 제한

- `playful` 또는 유사 옵션은 실험 기능으로만 제공한다.
- 민감 키워드 또는 위기 감정 감지 시 자동 비활성화된다.

수용 기준:
- 안전 차단 시 서버는 응답 메타에 `tone_fallback_applied: true`를 포함한다.

### FR-AIP-004 그림일기 카드 생성

- 사용자는 일기 상세 또는 분석 결과에서 `그림일기처럼 보기`를 요청할 수 있어야 한다.
- 서버는 텍스트 기반 카드 JSON을 반환한다.

수용 기준:
- 응답에는 `title`, `panels`, `dominant_emotion`, `closing_reflection`이 포함된다.
- 장면 수는 항상 3개다.

## 7. UX 제안

### 프로필 설정

섹션명:
- `AI 응답 스타일`
- `대화 스타일`

라벨 예시:
- 말투: `따뜻하게`, `차분하게`, `담백하게`, `유쾌하게(실험)`
- 응답 길이: `짧게`, `보통`, `조금 자세히`
- 조언 강도: `공감 위주`, `균형 있게`, `조언 조금 더`
- 대화 스타일: `따뜻한 친구형`, `차분한 코치형`, `담백한 정리형`

보조 문구:
- `감정 분석 기준은 바뀌지 않고, 표현 방식만 달라져요.`
- `유쾌한 톤은 일부 민감한 기록에서 자동으로 꺼질 수 있어요.`

### 그림일기 카드

CTA 예시:
- `오늘 하루를 그림일기처럼 정리해볼까요?`

안내 문구:
- `AI가 오늘의 분위기를 3컷 그림일기처럼 정리해드려요.`

## 8. 데이터 및 KPI

핵심 지표:
- 설정 저장률
- 설정별 재선택률
- 분석 응답 만족도
- 유쾌한 톤 사용 후 이탈률 변화
- 그림일기 카드 생성률 및 공유율

실험 기준:
- 2주 내 프로필 설정 진입 사용자 중 30% 이상이 새 옵션 저장
- 설정 저장 사용자군의 7일 재방문율이 비설정군 대비 개선

## 10. 리스크

- 옵션 과다로 인한 결정 피로
- 유쾌한 톤의 안전성 문제
- 페르소나가 상담 역할극처럼 오해될 위험
- 그림일기 카드가 실제 사실 단정처럼 보일 위험

대응:
- 프리셋 개수 제한
- 서버 측 fallback 강제
- UI에서 표현 방식 변경임을 명확히 고지
- 그림일기 카드에 분위기 요약임을 명시
